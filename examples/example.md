# TTY::Markdown

**tty-markdown** converts markdown document into a terminal friendly output.

## Examples

### Nested list items

- Item 1
  - Item 2
  - Item 3
    - Item 4
    - Item 5
      - Item 6
- Item 7

### Quote

>  Oh, you can put Markdown into a blockquote.

### Ruby syntax

```ruby
def hello(name)
  puts "Hello #{name}"
end
```
